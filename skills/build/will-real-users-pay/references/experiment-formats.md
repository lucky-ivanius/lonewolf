# Pricing experiment formats

Five cheap formats. Pick one based on cost, signal strength, and the audience you can actually reach in 14 days.

## 1. Landing page + waitlist

A single static page with the value prop, the price, and a "join waitlist" button. Drive traffic from one channel.

- Cost: low. A few hours to build, optional ad spend.
- Signal: medium. Conversion rate on the waitlist click is the proxy. Below 5% is a soft fail. Above 15% is a soft pass.
- When to use: B2C, when you have an existing audience or can buy targeted ads.
- Watch out: traffic without intent is noise. A retweet from a friend who has no payer intent does not count.

## 2. DM outreach

Send a short, direct message to 5 to 10 named candidates. Ask if they would pay the price for the offer.

- Cost: low. Time-bounded by how many DMs you can send personally.
- Signal: high. Each response is a real human with a name.
- When to use: B2B, pro user, niche communities (Telegram groups, Discord, Farcaster).
- Watch out: do not message friends. Candidates must be solving the problem the product solves.

## 3. Poll

Public poll on Twitter, Telegram, or a Discord channel where the target user already gathers.

- Cost: lowest.
- Signal: lowest. Polls capture aspirational answers, not commitment.
- When to use: only as a paired follow-up to DM outreach. The poll widens the funnel; the DM gets the real signal.
- Watch out: never use a poll alone. "I would pay" in a poll is not "I will pay".

## 4. Concierge offer

Offer to do the work manually for the price, today. The product does not need to exist.

- Cost: medium. The user manually delivers the value.
- Signal: highest. Anyone who pays for the manual version will pay for the automated version.
- When to use: when the value is concrete (e.g. a report, an audit, a setup, a managed service).
- Watch out: only valid if the manual offer maps cleanly to the future product.

## 5. Pre-order

Live payment page with a real charge. Refundable in writing if the product does not ship by date X.

- Cost: medium. Requires a payment processor and a refund commitment.
- Signal: highest. Money changing hands is the strongest commitment.
- When to use: when the product roadmap is firm and the user can credibly ship.
- Watch out: legally a pre-order is a contract. Be honest about the ship date and refund terms.

## How to pick

Default to DM outreach for the first round. It is cheap, it is high-signal, and it forces the user to talk to candidates by name.

Concierge or pre-order if the product is concrete enough to deliver manually or charge for in advance.

Landing page + waitlist if there is already an audience to drive.

Polls only as the front of a funnel that ends in DMs.
